
# MedallionTracker (ExileCore2)

Lightweight ExileCore2 plugin that tracks Incursion Medallions in Vaal Temple with color-coded guidance and per-medallion filtering.

**Author:** Strafeous  
**Version:** 1.0.0

---

## Features

- Name-based Incursion Medallion detection
- Per-medallion enable/disable toggles
- Unique color-coded beam per medallion
- Optional segmented path rendering
- Auto-clears after pickup
- Lightweight and dependency-free

---

## Installation

1. Extract the `MedallionTracker` folder into:
   ```
   ExileCore2/Plugins/Source/
   ```
2. Build using your ExileCore2 environment.
3. Enable the plugin via F12 menu.

---

## Supported Medallions

- Juatalotli (Red)
- Quipolatl (Lime)
- Xopec (Cyan)
- Estazunti (Purple)
- Zantipi (Yellow)
- Hayoxi (Orange)
- Uromoti (White)
- Azcapa (Aqua)
- Puhuarte (DeepPink)
